# PokerTrainer

Локальный покерный тренажёр для изучения GTO-стратегий.

## Tech Stack
- React + TypeScript
- Vite
- TailwindCSS + shadcn/ui
- Zustand
- TanStack Query
- Recharts

## Настройка и запуск

1. Установите зависимости (если используете npm):
```sh
npm install
```

2. Запустите dev-сервер:
```sh
npm run dev
```

Приложение будет доступно по адресу `http://localhost:8080` (или другой свободный порт).
